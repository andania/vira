# VIRAZ Platform

A revolutionary engagement economy platform connecting brands with users through interactive advertising.

## Technology Stack
- **Backend**: Node.js + Express + TypeScript
- **Frontend**: React + TypeScript
- **Database**: PostgreSQL + Prisma
- **Caching**: Redis
- **Real-time**: Socket.io
- **Container**: Docker

## Project Structure
See the generated folder structure for complete organization.

## Getting Started
1. Install dependencies: `pnpm install`
2. Set up environment: `cp .env.example .env`
3. Start databases: `docker-compose up -d`
4. Run migrations: `cd packages/backend && pnpm prisma:migrate`
5. Start development: `pnpm dev`
